How it Works

You curate the sources. The LLM does everything else — reading, summarizing, organizing, linking, and answering your questions.

Traditional PKM

  • Human reads the source
  • Human writes summaries
  • Human organizes and tags
  • Human maintains links
  • Human searches manually

Pensieve

  • Human curates sources
  • LLM summarizes everything
  • LLM writes wiki articles
  • LLM builds index and graph
  • LLM answers your questions

What you get

Everything you need to build a living, growing knowledge base powered by local LLMs.

📥

Smart Ingest

Ingest URLs, PDFs, markdown files, and entire Git repos. Auto-downloads images and generates metadata.

⚙

6-Step Compiler

Summarize → Extract concepts → Write articles → Build index → Glossary → Concept graph. Fully automated.

💬

Q&A Engine

Ask natural language questions. Get answers as text, research reports, or Marp slide decks.

🔍

Search

Keyword and semantic search with a built-in web UI. Cached indexes for instant results.

🛠

Wiki Linter

Find broken links, orphaned articles, missing frontmatter. Auto-fix issues and get LLM-powered improvement suggestions.

🌐

100% Local

Runs entirely on your machine with Ollama. No data leaves your computer. Cloud APIs available as optional fallback.

💎

Obsidian Native

Browse your wiki in Obsidian with graph view, Dataview dashboard, and Marp slides. Beautiful by default.

🔄

Feedback Loop

Query results file back into the wiki. Your explorations compound — the knowledge base grows with every question.

Quick Start

From zero to a working knowledge base in under 5 minutes.

# Clone and set up git clone https://github.com/ceparadise168/Pensieve.git cd Pensieve bash scripts/bootstrap.sh # Start Ollama ollama serve # Ingest your first source ./tools/kb ingest url "https://example.com/article" # Compile the wiki ./tools/kb compile --full # Open in Obsidian — select this directory as vault # Ask questions ./tools/kb query "What are the key concepts?" # Generate a presentation ./tools/kb query "Overview of main themes" --output slides
