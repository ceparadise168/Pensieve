Process
運作方式

You curate. The LLM compiles.

你策展，LLM 編譯。

Raw sources go in. A structured, interlinked wiki comes out. You never touch the wiki directly — it belongs to the LLM.

原始資料進去，結構化、互相連結的 wiki 出來。你不直接編輯 wiki — 那是 LLM 的領域。

Capabilities
功能特色

Everything the basin holds

冥想盆所承載的一切

A complete system for building living knowledge from raw sources.

從原始資料建構活知識的完整系統。

01

Smart Ingest

智慧攝入

URLs, PDFs, markdown, Git repos. Images auto-downloaded, metadata auto-generated.

網頁、PDF、Markdown、Git 儲存庫。圖片自動下載，metadata 自動生成。

02

6-Step Compiler

六步驟編譯器

Summarize, extract concepts, write articles, build index, glossary, concept graph.

摘要、提取概念、撰寫文章、建立索引、詞彙表、概念圖譜。

03

Q&A Engine

問答引擎

Natural language questions answered as text, research reports, or Marp slides.

自然語言提問，以文字、研究報告或 Marp 簡報回答。

04

Search

搜尋引擎

Keyword and semantic search with built-in web UI. Cached for instant results.

關鍵字與語意搜尋，內建 Web UI，索引快取即時回應。

05

Wiki Linter

Wiki 檢查器

Broken links, orphans, missing frontmatter. Auto-fix and LLM improvement suggestions.

斷裂連結、孤立文章、缺失 frontmatter。自動修復及 LLM 改善建議。

06

100% Local

完全本地運行

Runs on your machine with Ollama. No data leaves. Cloud APIs as optional fallback.

透過 Ollama 在你的電腦上運行。資料不外流。雲端 API 可選備援。

07

Obsidian Native

Obsidian 原生

Graph view, Dataview dashboard, Marp slides. Browse your wiki beautifully.

圖譜視圖、Dataview 儀表板、Marp 簡報。美觀瀏覽你的 wiki。

08

Feedback Loop

回饋迴圈

Query results file back into the wiki. Knowledge compounds with every question.

查詢結果回填 wiki。知識隨每次提問持續累積成長。

Getting Started
快速開始

Five minutes to your first wiki

五分鐘建立你的第一個 Wiki

# Clone and set up git clone https://github.com/ceparadise168/Pensieve.git cd Pensieve && bash scripts/bootstrap.sh # Start Ollama ollama serve # Ingest your first source ./tools/kb ingest url "https://example.com/article" # Compile the wiki ./tools/kb compile --full # Ask questions ./tools/kb query "What are the key concepts?" # Generate slides ./tools/kb query "Overview of main themes" --output slides

Recommended models

推薦模型

Local inference on Apple Silicon via Ollama. No GPU required.

在 Apple Silicon 上透過 Ollama 本地推理，無需獨立 GPU。
